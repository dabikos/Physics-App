                    ЗАБЫТЫЕ ТЕМЫ ФИЗИКИ ДЛЯ ПРИЛОЖЕНИЯ
═══════════════════════════════════════════════════════════════════════════

1. ГРАВИТАЦИЯ (Раздел Механики)
   ├─ Закон всемирного тяготения
   ├─ Гравитационное поле и напряженность
   ├─ Вес и невесомость
   ├─ Искусственные спутники и орбиты
   ├─ Законы Кеплера
   ├─ Первая, вторая и третья космические скорости
   └─ Гравитационная потенциальная энергия

2. ГИДРОСТАТИКА И ГИДРОДИНАМИКА (Отдельный раздел или в Механику)
   ├─ Давление в жидкостях
   ├─ Закон Паскаля
   ├─ Закон Архимеда
   ├─ Плавание и погружение тел
   ├─ Атмосферное давление
   ├─ Барометр и манометр
   ├─ Гидравлические машины
   ├─ Уравнение неразрывности (течение жидкостей)
   ├─ Уравнение Бернулли
   ├─ Вязкость жидкостей
   └─ Формула Стокса (сила сопротивления)

3. АКУСТИКА (Раздел Волн)
   ├─ Звуковые волны
   ├─ Скорость звука в разных средах
   ├─ Громкость и интенсивность звука
   ├─ Децибелы (шкала громкости)
   ├─ Эффект Доплера
   ├─ Резонанс в акустике
   ├─ Ультразвук и инфразвук
   ├─ Тембр и музыкальные инструменты
   └─ Отражение и поглощение звука

4. СПЕЦИАЛЬНАЯ ТЕОРИЯ ОТНОСИТЕЛЬНОСТИ (СТО)
   ├─ Постулаты Эйнштейна
   ├─ Преобразования Лоренца
   ├─ Замедление времени (time dilation)
   ├─ Сокращение длины (length contraction)
   ├─ Инвариантность интервала
   ├─ Формула E = mc²
   ├─ Релятивистская энергия и импульс
   ├─ Релятивистическая динамика
   └─ Недостижимость скорости света

5. АСТРОНОМИЯ (Отдельный раздел или как подраздел Механики)
   ├─ НЕБЕСНАЯ МЕХАНИКА
   │  ├─ Солнечная система (строение)
   │  ├─ Орбиты планет и спутников
   │  ├─ Масса небесных тел
   │  ├─ Приливы и отливы
   │  └─ Затмения (Солнца и Луны)
   ├─ СОЛНЦЕ И ЗВЁЗДЫ
   │  ├─ Строение Солнца
   │  ├─ Солнечная активность (пятна, вспышки, протуберанцы)
   │  ├─ Спектры звёзд
   │  ├─ Диаграмма Герцшпрунга-Рассела
   │  ├─ Эволюция звёзд (от рождения к смерти)
   │  ├─ Типы звёзд (красные гиганты, белые карлики, нейтронные звёзды)
   │  ├─ Пульсары и магнетары
   │  └─ Чёрные дыры
   ├─ ГАЛАКТИКИ И КОСМОЛОГИЯ
   │  ├─ Млечный путь (строение, размер)
   │  ├─ Типы галактик (спиральные, эллиптические, булганные)
   │  ├─ Скопления галактик и Местное скопление
   │  ├─ Расширение Вселенной (закон Хаббла)
   │  ├─ Большой взрыв (Big Bang)
   │  ├─ Реликтовое излучение (CMB)
   │  ├─ Возраст и эволюция Вселенной
   │  ├─ Тёмная материя и тёмная энергия
   │  └─ Общая теория относительности (кратко)
   ├─ НАБЛЮДАТЕЛЬНАЯ АСТРОНОМИЯ
   │  ├─ Телескопы (рефрактор, рефлектор, радиотелескопы)
   │  ├─ Увеличение и разрешающая способность
   │  ├─ Спектроскопия небесных объектов
   │  ├─ Фотометрия
   │  ├─ Определение расстояний (параллакс, красное смещение)
   │  └─ Радиоастрономия
   └─ ПЛАНЕТЫ И ИХ СПУТНИКИ
      ├─ Характеристики планет Земной группы
      ├─ Характеристики газовых гигантов
      ├─ Луна (фазы, либрация, кратеры)
      ├─ Спутники планет (Титан, Ганимед и т.д.)
      ├─ Астероиды и пояс Койпера
      ├─ Кометы и метеориты
      ├─ Экзопланеты (планеты вокруг других звёзд)
      └─ Поиск жизни (SETI, habitable zone)

ПРИОРИТЕТ ДОБАВЛЕНИЯ:

   1. Гравитация (9-10 класс школы)
   2. Гидростатика + Архимед (7-10 класс школы)
   3. Акустика (звук) (7-9 класс школы)

🟡 ВАЖНЫЕ (старшие классы, углубленные):
   4. СТО (11 класс, продвинутые школы)
   5. Астрономия (10-11 класс)
