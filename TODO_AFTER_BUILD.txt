1. СОБРАТЬ DEV BUILD (push-уведомления не работают в Expo Go)
--------------------------------------------------------------

   cd "D:\Physics App\Physics-App\frontend"

   # Установить EAS CLI (если ещё нет)
   npm install -g eas-cli

   # Войти в Expo аккаунт
   eas login

   # Собрать dev build для Android
   eas build --profile development --platform android

   # После сборки скачать .apk с сайта expo.dev и установить на телефон
   # Запуск приложения:
   npx expo start --dev-client


2. PRODUCTION BUILD (для Google Play)
--------------------------------------

   # APK для тестирования
   eas build --profile preview --platform android

   # AAB для Google Play
   eas build --profile production --platform android


3. НАСТРОИТЬ CRON ДЛЯ ЕЖЕДНЕВНЫХ PUSH-НАПОМИНАНИЙ
----------------------------------------------------

   Бэкенд endpoint: POST /cron/daily-push
   Отправляет push неактивным пользователям (стрик-напоминания)

   Шаги:
   a) Зайди на https://cron-job.org и зарегистрируйся
   b) Создай новую задачу:
      - URL:     https://physics-app-hgw72.ondigitalocean.app/api/cron/daily-push
      - Method:  POST
      - Headers: X-Cron-Secret: physics-ai-cron-2024
      - Schedule: 0 14 * * *  (каждый день в 14:00 UTC = 20:00 Астана)
   c) Сохрани и протестируй (кнопка "Run now")

   Альтернативное время:
   - 0 4 * * *   = 10:00 Астана (утром перед школой)
   - 0 10 * * *  = 16:00 Астана (после школы)
   - 0 14 * * *  = 20:00 Астана (вечером)


4. ПРОВЕРКА ЧТО ВСЁ РАБОТАЕТ
------------------------------

   a) Войти в приложение на телефоне (dev build)
   b) Профиль → Уведомления → должен появиться список
   c) Попросить учителя назначить тест → ученик получит push
   d) Ученик сдаёт тест → учитель получит push с результатом
   e) Не заходить 1 день → cron отправит напоминание
