В) F * S * cos(α)
С) m * v^2 / 2
D) N * t

В) Ватт (Вт)
С) Джоуль (Дж)

А) m * g * h
В) Максимальному значению
С) Нулю
D) Зависит от формы траектории
Правильный ответ: С

4. В каком случае сила не совершает механической работы (A = 0)?
А) Сила совпадает по направлению с перемещением.
В) Сила направлена противоположно перемещению.

D) Произведение массы на скорость.
Правильный ответ: А

Задача 1. Автомобиль движется прямолинейно. Сила тяги двигателя 2000 Н. Какую работу совершает двигатель при перемещении автомобиля на 100 м?
Дано:
F = 2000 Н
S = 100 м
α = 0° (направления силы и перемещения совпадают)
------------
Найти: A - ?

Решение:
1. Формула работы: A = F * S * cos(α).
2. Так как cos(0°) = 1, A = 2000 * 100 = 200000 Дж = 200 кДж.

Ответ: A = 200 кДж.


Задача 2. Мальчик тянет санки с силой 50 Н под углом 60° к горизонту. Расстояние равно 20 м. Вычислите работу силы.
Дано:
F = 50 Н
S = 20 м
α = 60°
------------
Найти: A - ?

Решение:
1. Формула работы: A = F * S * cos(α).
2. Подстановка: A = 50 * 20 * cos(60°).
3. cos(60°) = 0.5.
4. A = 1000 * 0.5 = 500 Дж.

Ответ: A = 500 Дж.


Задача 3. Грузчик поднимает ящик массой 20 кг на высоту 1.5 м. Какую работу совершает грузчик, если он поднимает груз равномерно? (g=10)
Дано:
m = 20 кг
h = 1.5 м
Ускорение a = 0.
------------
Найти: A - ?

Решение:
1. При равномерном подъеме сила грузчика F равна силе тяжести F_т = m*g.
2. перемещение S = h.
3. Сила направлена вверх, перемещение тоже вверх (α = 0°).
4. A = F * S = m*g*h.
5. A = 20 * 10 * 1.5 = 200 * 1.5 = 300 Дж.

Ответ: A = 300 Дж.


Задача 4. Камень массой 3 кг падает в шахту. Какую работу совершит сила тяжести за первые 2 секунды свободного падения без начальной скорости? (g=10)
Дано:
m = 3 кг
t = 2 с
v0 = 0
g = 10 м/с^2
------------
Найти: A_т - ?

Решение:
1. Найдем путь: h = (g * t^2) / 2 = (10 * 2^2) / 2 = 10 * 4 / 2 = 20 м.
2. Работа силы тяжести: A = F_т * h = m * g * h.
3. A = 3 * 10 * 20 = 600 Дж.

Ответ: Работа силы тяжести равна 600 Дж.


Задача 5. Кран поднял груз весом 5000 Н на высоту 10 м за 20 секунд. Определите мощность подъемного механизма крана.
Дано:
P = 5000 Н => F = 5000 Н (сила тяги)
h = 10 м => S = 10 м
t = 20 с
------------
Найти: N - ?

Решение:
1. Работа, совершенная краном: A = F * S = 5000 * 10 = 50000 Дж.
2. Формула мощности: N = A / t.
3. N = 50000 / 20 = 2500 Вт = 2.5 кВт.

Ответ: N = 2500 Вт (2.5 кВт).

В) Кинетическая энергия всегда сохраняется.
С) Кинетическая энергия может быть отрицательной.
D) Сумма кинетической и потенциальной энергии всегда равна нулю.

С) Увеличится в 9 раз

Правильный ответ: С

А) Да, если тело движется в обратном направлении.

--- ЗАДАЧИ ---
Задача 1. Автомобиль массой 1200 кг движется со скоростью 15 м/с. Чему равна его кинетическая энергия?
Дано:
m = 1200 кг
v = 15 м/с
------------
Найти: E_к - ?

Решение:
1. Формула кинетической энергии: E_к = (m * v^2) / 2.
2. Подставляем значения: E_к = 1200 * 15^2 / 2 = 1200 * 225 / 2 = 600 * 225 = 135000 Дж = 135 кДж.

Ответ: E_к = 135 кДж.


Задача 2. Пуля массой 9 г пробивает доску. Ее скорость при входе 800 м/с, а при выходе 400 м/с. Какую работу совершили силы сопротивления доски?
Дано:
m = 9 г = 0.009 кг
v1 = 800 м/с
v2 = 400 м/с
------------
Найти: A_сопр - ?

Решение:
1. По теореме о кинетической энергии: A_сопр = E_к2 - E_к1.
2. E_к1 = (0.009 * 800^2) / 2 = (0.009 * 640000) / 2 = 5760 / 2 = 2880 Дж.
3. E_к2 = (0.009 * 400^2) / 2 = (0.009 * 160000) / 2 = 1440 / 2 = 720 Дж.
4. A_сопр = 720 - 2880 = -2160 Дж (работа отрицательна, так как сила сопротивления направлена против движения).

Ответ: Работа сил сопротивления равна -2160 Дж.


Задача 3. Для разгона машины массой 1000 кг из состояния покоя совершили работу в 200 кДж (пренебрегая потерями). Какую скорость приобрела машина?
Дано:
m = 1000 кг
A = 200 кДж = 200000 Дж
v0 = 0
E_к1 = 0
------------
Найти: v - ?

Решение:
1. По теореме о кинетической энергии: A = E_к2 - E_к1 = E_к2.
2. E_к2 = (m * v^2) / 2 = 200000.
3. v^2 = (2 * A) / m = (2 * 200000) / 1000 = 400000 / 1000 = 400.
4. v = sqrt(400) = 20 м/с.

Ответ: v = 20 м/с.


Задача 4. Камень брошен вертикально вверх со скоростью 10 м/с. На какой высоте кинетическая энергия камня станет равной нулю? (g=10, сопротивлением пренебречь, через кинематику)
Дано:
v0 = 10 м/с
v = 0 (на высоте E_k = 0)
g = 10 м/с^2
------------
Найти: h - ?

Решение:
1. Удобно использовать формулу без времени: 2*a*S = v^2 - v0^2.
2. При движении вверх a = -g = -10, v = 0.
3. -2 * 10 * h = 0 - 100 => -20 * h = -100.
4. h = 5 м.

Ответ: h = 5 м.


Задача 5. Если импульс тела увеличить в 2 раза, оставив массу постоянной, во сколько раз изменится его кинетическая энергия?
Дано:
p2 = 2 * p1
m2 = m1 = m
------------
Найти: E_к2 / E_к1 - ?

Решение:
1. Связь кинетической энергии и импульса: E_к = p^2 / (2m).
2. E_к1 = p1^2 / (2m).
3. E_к2 = (2*p1)^2 / (2m) = 4 * p1^2 / (2m) = 4 * E_к1.
4. Таким образом, кинетическая энергия увеличится в 4 раза.

Ответ: Кинетическая энергия увеличится в 4 раза.

Задача 1. Мяч массой 100 г бросили вертикально вверх со скоростью 20 м/с. На какую максимальную высоту поднимется мяч? Трение не учитывать. (g=10)
Дано:
v0 = 20 м/с
v = 0 (в верхней точке)
g = 10 м/с^2
------------
Найти: h_max - ?

Решение:
1. По закону сохранения механической энергии: E_к0 + E_p0 = E_к_top + E_p_top.
2. Начальный уровень (h=0) -> E_p0 = 0. E_к0 = m*v0^2 / 2.
3. В верхней точке (v=0) -> E_к_top = 0. E_p_top = m*g*h_max.
4. Отсюда: m*v0^2 / 2 = m*g*h_max. Масса сокращается.
5. h_max = v0^2 / (2*g) = 20^2 / (2*10) = 400 / 20 = 20 м.

Ответ: h_max = 20 м.


Задача 2. Камешек свободно падает с высоты 45 м. Какой скорости он достигнет у поверхности земли? (g=10)
Дано:
h0 = 45 м
v0 = 0
g = 10 м/с^2
------------
Найти: v - ?

Решение:
1. ЗСЭ: E_p_нач = E_к_кон.
2. m*g*h0 = (m * v^2) / 2.
3. v^2 = 2 * g * h0 = 2 * 10 * 45 = 900.
4. v = sqrt(900) = 30 м/с.

Ответ: v = 30 м/с.


Задача 3. Пружинный пистолет заряжается. Пружину с жесткостью 1000 Н/м сжали на 5 см. Пуля массой 10 г вылетает горизонтально. Определить скорость вылета пули.
Дано:
k = 1000 Н/м
x = 5 см = 0.05 м
m = 10 г = 0.01 кг
------------
Найти: v - ?

Решение:
1. ЗСЭ: Потенциальная энергия упруго деформированной пружины полностью переходит в кинетическую энергию пули в момент отрыва.
2. (k * x^2) / 2 = (m * v^2) / 2.
3. k * x^2 = m * v^2 => v^2 = (k * x^2) / m.
4. v^2 = (1000 * 0.05^2) / 0.01 = (1000 * 0.0025) / 0.01 = 2.5 / 0.01 = 250.
5. v = sqrt(250) ≈ 15.8 м/с.

Ответ: Скорость вылета ≈ 15.8 м/с.


Задача 4. Мальчик съезжает на санках с горы высотой 5 м. У основания горы скорость санок равна 6 м/с. Масса мальчика с санками 50 кг. Какая работа была совершена силами трения на этом пути? (g=10)
Дано:
h = 5 м
v0 = 0
v = 6 м/с
m = 50 кг
g = 10 м/с^2
------------
Найти: A_тр - ?

Решение:
1. Закон изменения мех. энергии: E_кон - E_нач = A_неконсервативных_сил (работа силы трения).
2. Начальная энергия (наверху): E_нач = m*g*h = 50 * 10 * 5 = 2500 Дж.
3. Конечная энергия (внизу): E_кон = (m*v^2)/2 = (50 * 6^2)/2 = 50 * 36 / 2 = 50 * 18 = 900 Дж.
4. Изменение энергии: ΔE = E_кон - E_нач = 900 - 2500 = -1600 Дж.
5. Значит, работа силы трения равна -1600 Дж.

Ответ: A_тр = -1600 Дж.


Задача 5. Шарик массой m привязан к нити длиной L = 1 м. Его отвели на угол 90° к вертикали (горизонтальное положение) и отпустили. Какова будет скорость шарика при прохождении нижней точки траектории?
Дано:
L = 1 м
α = 90°
v0 = 0
h0 = L (относительно нижней точки)
------------
Найти: v - ?

Решение:
1. Обозначим за h=0 нижнюю точку. Начальная высота шарика h0 = L. E_нач = mgL. 
2. В нижней точке высота 0, кинетическая энергия E_кон = (m*v^2)/2.
3. По ЗСЭ: mgL = (m*v^2)/2.
4. v = sqrt(2*g*L) = sqrt(2 * 10 * 1) = sqrt(20) ≈ 4.47 м/с.

Ответ: v ≈ 4.47 м/с.
