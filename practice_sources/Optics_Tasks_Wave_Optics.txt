РАЗДЕЛ: Оптика
ПОДРАЗДЕЛ: Волновая оптика

=== ТЕМА: interference (Интерференция) ===

--- ТЕСТЫ ---
1. Интерференция света возникает при сложении:
А) когерентных световых волн
В) любых тел без волн
С) постоянных токов
D) молекул газа
Правильный ответ: А

2. Условие интерференционного максимума:
А) Δ = kλ
В) Δ = (2k+1)λ/2
С) Δ = λ/4 всегда
D) Δ = 0 только при k = 5
Правильный ответ: А

3. Условие интерференционного минимума:
А) Δ = (2k+1)λ/2
В) Δ = kλ
С) Δ = λL/d
D) Δ = c/n
Правильный ответ: А

4. В опыте Юнга расстояние между соседними светлыми полосами:
А) Δx = λL/d
В) Δx = d/(λL)
С) Δx = L/(λd)
D) Δx = λd/L
Правильный ответ: А

5. Если расстояние между щелями d увеличить, расстояние между интерференционными полосами:
А) уменьшится
В) увеличится
С) не изменится
D) станет равным длине волны
Правильный ответ: А

--- ЗАДАЧИ ---
Задача 1. В опыте Юнга λ = 600 нм, L = 2 м, d = 0,4 мм. Найдите расстояние между соседними полосами.
Дано:
λ = 600 нм = 600×10⁻⁹ м
L = 2 м
d = 0,4 мм = 0,4×10⁻³ м
------------
Найти: Δx - ?

Решение:
Δx = λL/d = 600×10⁻⁹ · 2 / (0,4×10⁻³) = 3×10⁻³ м.

Ответ: 3 мм.

Задача 2. Разность хода двух волн равна 1,5 мкм, длина волны 500 нм. Будет максимум или минимум?
Дано:
Δ = 1,5 мкм = 1500 нм
λ = 500 нм
------------
Найти: тип интерференции - ?

Решение:
Δ/λ = 1500/500 = 3.
Разность хода равна целому числу длин волн, значит Δ = kλ.

Ответ: максимум, k = 3.

Задача 3. Для света λ = 400 нм найдите первую разность хода, при которой наблюдается минимум.
Дано:
λ = 400 нм
k = 0
------------
Найти: Δmin - ?

Решение:
Для минимума Δ = (2k+1)λ/2.
При k = 0: Δ = λ/2 = 200 нм.

Ответ: 200 нм.

Задача 4. В опыте Юнга расстояние между полосами 2 мм, экран на расстоянии 1,5 м, λ = 500 нм. Найдите расстояние между щелями.
Дано:
Δx = 2 мм = 2×10⁻³ м
L = 1,5 м
λ = 500 нм = 500×10⁻⁹ м
------------
Найти: d - ?

Решение:
Δx = λL/d, значит d = λL/Δx.
d = 500×10⁻⁹ · 1,5 / 2×10⁻³ = 3,75×10⁻⁴ м.

Ответ: 0,375 мм.

Задача 5. Почему для устойчивой интерференционной картины нужны когерентные источники?
Дано:
интерференция света
------------
Найти: роль когерентности - ?

Решение:
Когерентные источники сохраняют постоянную разность фаз.
Тогда максимумы и минимумы остаются в одних местах, и картина не смазывается во времени.

Ответ: когерентность нужна для устойчивой разности фаз и стабильных полос.

=== ТЕМА: diffraction (Дифракция) ===

--- ТЕСТЫ ---
1. Дифракция — это:
А) огибание волной препятствий и отклонение от прямолинейного распространения
В) только отражение от зеркала
С) только нагревание света
D) движение электронов в проводнике
Правильный ответ: А

2. Условие максимумов для дифракционной решётки:
А) d·sinφ = kλ
В) d/sinφ = kλ
С) sinφ = kd/λ
D) d = λ/sinφ при любом k = 0
Правильный ответ: А

3. Чем меньше размер щели по сравнению с длиной волны, тем дифракция:
А) заметнее
В) слабее
С) полностью исчезает
D) превращается в преломление
Правильный ответ: А

4. Для центрального максимума решётки порядок k равен:
А) 0
В) 1
С) 2
D) -5
Правильный ответ: А

5. Если длина волны увеличивается при той же решётке и том же k, угол дифракционного максимума:
А) увеличивается
В) уменьшается
С) не меняется
D) всегда равен 90°
Правильный ответ: А

--- ЗАДАЧИ ---
Задача 1. Дифракционная решётка имеет период d = 2 мкм. Свет с λ = 500 нм даёт максимум первого порядка. Найдите sinφ.
Дано:
d = 2 мкм = 2×10⁻⁶ м
λ = 500 нм = 5×10⁻⁷ м
k = 1
------------
Найти: sinφ - ?

Решение:
d·sinφ = kλ.
sinφ = kλ/d = 5×10⁻⁷ / 2×10⁻⁶ = 0,25.

Ответ: sinφ = 0,25.

Задача 2. Для решётки d = 1,5 мкм максимум второго порядка наблюдается при sinφ = 0,8. Найдите длину волны.
Дано:
d = 1,5 мкм
k = 2
sinφ = 0,8
------------
Найти: λ - ?

Решение:
λ = d·sinφ/k = 1,5 · 0,8 / 2 = 0,6 мкм.

Ответ: 0,6 мкм = 600 нм.

Задача 3. Может ли решётка с d = 1 мкм дать максимум третьего порядка для λ = 500 нм?
Дано:
d = 1 мкм
λ = 0,5 мкм
k = 3
------------
Найти: возможен ли максимум - ?

Решение:
sinφ = kλ/d = 3 · 0,5 / 1 = 1,5.
Так как sinφ не может быть больше 1, такой максимум невозможен.

Ответ: нет, максимум третьего порядка невозможен.

Задача 4. Почему компакт-диск разлагает белый свет на цвета?
Дано:
на поверхности диска есть частые дорожки
белый свет содержит разные λ
------------
Найти: причина цветных полос - ?

Решение:
Дорожки диска работают как дифракционная решётка.
Для разных длин волн условие d·sinφ = kλ выполняется при разных углах, поэтому цвета расходятся.

Ответ: из-за дифракции на регулярных дорожках диска.

Задача 5. Решётка даёт первый максимум для красного света под большим углом, чем для синего. Что можно сказать о длинах волн?
Дано:
k одинаковый
d одинаковый
φкрасн > φсин
------------
Найти: сравнение λ - ?

Решение:
Из d·sinφ = kλ следует, что при одинаковых d и k длина волны пропорциональна sinφ.
Больший угол означает большую длину волны.

Ответ: у красного света длина волны больше, чем у синего.

=== ТЕМА: polarization (Поляризация) ===

--- ТЕСТЫ ---
1. Поляризация показывает, что световая волна:
А) поперечная
В) продольная
С) не является волной
D) не переносит энергию
Правильный ответ: А

2. Закон Малюса:
А) I = I₀cos²φ
В) I = I₀sinφ
С) I = I₀/φ
D) I = φ/I₀
Правильный ответ: А

3. Если угол между осями поляризаторов 90°, интенсивность после второго идеального поляризатора:
А) равна нулю
В) максимальна
С) равна I₀
D) всегда равна половине
Правильный ответ: А

4. При φ = 0° по закону Малюса:
А) I = I₀
В) I = 0
С) I = I₀/2
D) I = 2I₀
Правильный ответ: А

5. Поляризационные очки уменьшают блики, потому что:
А) пропускают преимущественно колебания одной ориентации
В) увеличивают скорость света
С) меняют массу фотонов
D) превращают свет в звук
Правильный ответ: А

--- ЗАДАЧИ ---
Задача 1. На анализатор падает поляризованный свет интенсивности 80 Вт/м². Угол между направлением поляризации и осью анализатора 60°. Найдите интенсивность после анализатора.
Дано:
I₀ = 80 Вт/м²
φ = 60°
------------
Найти: I - ?

Решение:
I = I₀cos²φ = 80 · cos²60° = 80 · 0,25 = 20 Вт/м².

Ответ: 20 Вт/м².

Задача 2. После анализатора осталось 25% начальной интенсивности поляризованного света. Найдите угол φ.
Дано:
I/I₀ = 0,25
------------
Найти: φ - ?

Решение:
I/I₀ = cos²φ = 0,25.
cosφ = 0,5, значит φ = 60°.

Ответ: 60°.

Задача 3. Два идеальных поляризатора скрещены под углом 90°. Почему свет почти не проходит?
Дано:
φ = 90°
------------
Найти: причина отсутствия света - ?

Решение:
По закону Малюса I = I₀cos²90°.
cos90° = 0, поэтому I = 0.

Ответ: второй поляризатор не пропускает колебания, перпендикулярные его оси.

Задача 4. Интенсивность поляризованного света 120 Вт/м². После анализатора она стала 90 Вт/м². Найдите cos²φ.
Дано:
I₀ = 120 Вт/м²
I = 90 Вт/м²
------------
Найти: cos²φ - ?

Решение:
I = I₀cos²φ.
cos²φ = I/I₀ = 90/120 = 0,75.

Ответ: cos²φ = 0,75.

Задача 5. Почему отражённый от воды свет часто частично поляризован?
Дано:
отражение от поверхности воды
------------
Найти: причина поляризации бликов - ?

Решение:
При отражении от диэлектрической поверхности разные направления колебаний отражаются неодинаково.
Поэтому в отражённом свете одна ориентация колебаний преобладает.

Ответ: отражение от воды частично упорядочивает направление колебаний электрического поля.
