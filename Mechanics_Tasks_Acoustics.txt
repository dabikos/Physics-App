Задача 1. Какова длина звуковой волны в воздухе, если частота колебаний источника составляет 440 Гц (нота Ля первой октавы), а скорость звука в воздухе 340 м/с?

ν = 440 Гц
v = 340 м/с
------------
Найти: λ - ?

Решение:
1. Формула скорости: v = λ * ν.
2. Длина волны: λ = v / ν.
3. λ = 340 / 440 = 34 / 44 ≈ 0.77 м.

Задача 3. Мальчик крикнул, стоя перед скалой, и услышал эхо через 3 секунды. На каком расстоянии от него находится скала? (v_звука = 340 м/с).

v = 340 м/с
------------
Найти: L - ?

Решение:
1. Звук проходит путь до скалы и обратно, то есть 2 * L = v * t_общ.
2. L = (v * t_общ) / 2.
3. L = (340 * 3) / 2 = 1020 / 2 = 510 м.

Ответ: Период равен 5 мс.

M_He = 4 г/моль

------------
Найти: v_He / v_Ar - ?

Решение:
1. Отношение скоростей: v_He / v_Ar = √((γ*R*T/M_He) / (γ*R*T/M_Ar)).

------------
Найти: Объяснение.

Решение:
1. Скорость распространения волны зависит только от свойств среды (для твердых тел E и ρ, для газов T и M).

I1 = 10⁻⁵ Вт/м²
I2 = 10⁻² Вт/м²
------------
Найти: ΔL - ?

Решение:
1. ΔL = L2 - L1 = 10 * lg(I2 / I0) - 10 * lg(I1 / I0) = 10 * lg(I2 / I1).
2. I2 / I1 = 10⁻² / 10⁻⁵ = 10³.
3. ΔL = 10 * lg(10³) = 10 * 3 = 30 дБ.

N = 2
L1 = 60 дБ
------------
Найти: L2 - ?

Решение:
1. Одному человеку (L1 = 60 дБ) соответствует интенсивность I1 = I0 * 10^(60/10) = 10⁶ * I0.
2. Для 2 человек интенсивности просто складываются: I_общ = 2 * I1 = 2 * 10⁶ * I0.
3. Уровень L_общ = 10 * lg(I_общ / I0) = 10 * lg(2 * 10⁶) = 10 * (lg(2) + 6) ≈ 10 * (0.3 + 6) = 63 дБ.

Ответ: Уровень составит примерно 63 дБ (удвоение интенсивности дает прибавку в 3 дБ).


Задача 5. Уровень шума от двигателя самолета снизили с 140 дБ до 120 дБ. Во сколько раз уменьшилась физическая мощность акустического излучения?

L2 = 120 дБ
------------
Найти: P1 / P2 - ?

Решение:

4. Отсюда P1 / P2 = 10² = 100.

Ответ: Физическая мощность шума уменьшилась в 100 раз.

Задача 1. Автомобиль движется к неподвижному наблюдателю со скоростью 34 м/с. Он издает звуковой сигнал постоянной частоты 1000 Гц. Какую частоту услышит наблюдатель? (Скорость звука в воздухе 340 м/с).

ν0 = 1000 Гц
v_зв = 340 м/с
v_набл = 0
------------
Найти: ν' - ?

Решение:
1. Формула эффекта Доплера при приближении источника к неподвижному наблюдателю:

2. ν' = 1000 * [ 340 / (340 - 34) ] = 1000 * (340 / 306).
3. 340 / 306 ≈ 1.111.
4. ν' = 1000 * 1.111 = 1111 Гц.

Задача 2. Тот же автомобиль удаляется от неподвижного наблюдателя со скоростью 34 м/с. Какую частоту услышит наблюдатель (исходная частота гудка 1000 Гц)?

v_ист = 34 м/с (удаляется)
ν0 = 1000 Гц
v_зв = 340 м/с

Решение:
1. Формула при удалении: ν' = ν0 * [ v_зв / (v_зв + v_ист) ].
2. ν' = 1000 * [ 340 / (340 + 34) ] = 1000 * (340 / 374).
3. 340 / 374 ≈ 0.909.
4. ν' = 1000 * 0.909 = 909 Гц.

v_ист = 30 м/с (движется НАВСТРЕЧУ)
ν0 = 500 Гц
v_зв = 340 м/с

Решение:
1. Общая формула Доплера: ν' = ν0 * [ (v_зв + v_набл) / (v_зв - v_ист) ].

2. ν' = 500 * [ (340 + 20) / (340 - 30) ] = 500 * (360 / 310) = 500 * (36 / 31).

Задача 4. Летучая мышь летит к стене со скоростью 5 м/с, издавая писк частотой 50 кГц. Какую частоту отраженного эха она воспримет? (Скорость звука 340 м/с).
Дано:
v = 5 м/с
ν0 = 50 кГц
v_зв = 340 м/с
------------
Найти: ν'' - ?

Решение:
1. Сначала стена (как неподвижный "наблюдатель") принимает частоту от приближающегося источника: 
   ν' = ν0 * [ v_зв / (v_зв - v) ].
2. Стена отражает звук этой же частоты ν' (теперь она источник).
3. Мышь теперь является приемником, летящим НАВСТРЕЧУ этому источнику-стене: 
   ν'' = ν' * [ (v_зв + v) / v_зв ].
4. Подставляем ν': ν'' = ν0 * [ (v_зв + v) / (v_зв - v) ].
5. ν'' = 50 * [ (340 + 5) / (340 - 5) ] = 50 * (345 / 335) = 50 * 1.0298 ≈ 51.5 кГц.

Ответ: Мышь услышит отраженный звук частотой 51.5 кГц.


Задача 5. Самолет летит на высоте H со сверхзвуковой скоростью числа Маха М (то есть v_сам = M * v_зв). Докажите геометрически или с помощью формул, какой угол α (полуугол при вершине) описывает граница конуса звуковой ударной волны (конус Маха). Вычислите этот угол для числа Маха М = 2.
Дано:
M = 2 (то есть v_сам = 2 * v_зв)
------------
Найти: α - ?

Решение:
1. Пока самолет проходит путь L = v_сам * t, звуковая волна, испущенная в начальной точке, проходит радиус R = v_зв * t.
2. В прямоугольном треугольнике, образованном траекторией полета и радиусом фронта волны: sin(α) = R / L.
3. sin(α) = (v_зв * t) / (v_сам * t) = v_зв / v_сам = 1 / M.
4. Для M = 2: sin(α) = 1/2 = 0.5.
5. Значит, угол α = arcsin(0.5) = 30°.

Ответ: Полуугол конуса Маха равен 30°.

Задача 3. Скорость распространения поперечной волны по струне задается формулой v = √(T/μ), где T - натяжение, μ - линейная плотность струны (кг/м). Тонкая струна гитары имеет длину L = 0.65 м, натянута силой 100 Н. Линейная плотность 0.001 кг/м. Вычислите частоту основного тона.
Дано:
L = 0.65 м
T = 100 Н
μ = 0.001 кг/м
------------
Найти: ν1 - ?

Решение:
1. Скорость волны на струне: v = √(100 / 0.001) = √(100000) ≈ 316 м/с.
2. В струне, закрепленной с двух концов (два узла), основная стоячая волна составляет полволны: L = λ1 / 2 => λ1 = 2 * L = 1.30 м.

Ответ: Частота основного тона около 243 Гц (это примерно нота Си).


Задача 4. Как музыкант на гитаре повышает высоту тона определенной струны во время игры (например, зажимая лад)?

------------

Ответ: Звук становится выше за счет уменьшения длины струны и, как следствие, уменьшения длины стоячей волны.

Задача 2. В большом концертном зале звук затухает (падает на 60 дБ от начального уровня) за 2 секунды. Это время называется временем реверберации T60. Сколько раз звук успеет отразиться от стен туда и обратно по длине зала L = 34 метра, пока он полностью не затухнет? (v = 340 м/с).
Дано:
T60 = 2 с
L = 34 м
v = 340 м/с
------------
Найти: N (количество полных проходов туда-обратно) - ?

Решение:
1. За время T60 звук проходит общий путь: S_общ = v * T60 = 340 * 2 = 680 м.
2. Один проход туда и обратно равен 2 * L = 68 м.
3. Количество проходов: N = S_общ / (2*L) = 680 / 68 = 10 раз.

Задача 4. Уровень интенсивности звука от одного динамика перед звукопоглощающей стеной (шторой) падает на 10 дБ при прохождении сквозь штору. Во сколько раз уменьшается интенсивность (мощность) звука?
Дано:
ΔL = -10 дБ
------------
Найти: I_выход / I_вход - ?

Решение:
1. Формула Децибел для затухания: ΔL = 10 * lg(I_выход / I_вход).
2. -10 = 10 * lg(I_вых / I_вх).
3. -1 = lg(I_вых / I_вх) => I_вых / I_вх = 10⁻¹ = 0.1 (или 1/10).

Ответ: Интенсивность уменьшается в 10 раз.
